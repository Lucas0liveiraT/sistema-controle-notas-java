# Sistema de Controle de Notas

Projeto desenvolvido em Java para praticar lógica de programação e estruturas de repetição.

## Funcionalidades

- Cadastro de alunos
- Cálculo da média da turma
- Identificação da maior e menor nota
- Contagem de alunos aprovados
- Exibição de relatório formatado

## Regras

- Nota mínima para aprovação: 6.0

## Tecnologias utilizadas

- Java
- Scanner
- Estruturas condicionais (if/else)
- Estruturas de repetição (for)
